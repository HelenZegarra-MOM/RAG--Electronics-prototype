raspberry pi pico datasheet 1. microcontroller specifications : type : rp2040 memory : 2mb flash, 264kb sram. clock speed : 133 mhz. flexible clock. peripherals : 2x uart, 2x spi, 2x i2c, 4x adc, 16x pwm controllers. 2. pin configurations : gpio pins : 26 ( most can be used as digital i / o ). analog input pins : 3 ( with dedicated adc ). power pins : vbus ( 5v from usb ), vsys ( external 1. 8 - 5. 5v ), 3. 3v, gnd. communication : all common serial interfaces. reset pin : run pin. 3. power requirements : operating voltage : 3. 3v. input voltage : 1. 8v - 5. 5v ( for vsys ). 4. programming interface : usb mass storage ( uf2 bootloader ). swd interface for debugging. 5. additional information : interrupts : flexible interrupt system. pwm channels : up to 16. refer to the official rp2040 datasheet for comprehensive details.
